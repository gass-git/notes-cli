## Install with Scoop

```sh
scoop install https://raw.githubusercontent.com/gass-git/notes-cli/refs/heads/main/notes-cli.json
```

## Commands

View all your notes:
```sh
notes-cli show
```

Add a new note:
```sh
notes-cli add "Your note here"
```

Delete note number n:
```sh
notes-cli delete n
```
